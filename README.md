Airline Review Sentiment Insight Platform
A cloud-based web application that enables users to share airline experiences and receive instant sentiment analysis (Positive / Neutral / Negative) in real time.
Home Page
The landing page introduces the platform with options for:
User Login
Admin Login
Registration
It provides a clean and attractive UI to start the user journey.

User Registration Page
A simple and modern User Registration Interface that allows users to create an account by providing:
Full Name
Email
Password
The design is minimal, responsive, and user-friendly.

Review Submission Module
This module allows users to:
Select an airline
Choose a review category (e.g., Food, Seat Comfort)
Write feedback in any language
The system processes the review and provides instant sentiment feedback.
<img width="858" height="823" alt="Review Submission" src="https://github.com/user-attachments/assets/b42cc925-43aa-412d-bf8c-5185426bd235" />
Email Notification (Sentiment Result)
After submitting a review, users receive an automatic email notification powered by AWS SNS (Simple Notification Service).
The email includes:
Airline Name
Category
Original Review
Translated Review
Detected Language
Sentiment Result
Timestamp

Admin Registration Module
This module allows administrators to securely register and manage the platform.
Admin capabilities include:
Monitoring user reviews
Analyzing sentiment results
Managing platform operations

Airline Review Analytics Dashboard
The Analytics Dashboard provides a centralized view of all airline reviews.
Key Features:
Sentiment summary (Positive / Negative / Neutral)
Filter by airline, category, and sentiment
Detailed review table with:
Original & translated reviews
Sentiment results
Timestamp
This helps admins gain real-time insights and make data-driven decisions.
